# 💇‍♀️ Salon Finder Web App

A modern and responsive Salon Finder Web Application built using React.js.  
This application allows users to explore beauty salons, search services, and save their favorite salons with a clean and user-friendly interface.

---

## 🚀 Live Demo
🔗 https://salon-finder-git-main-astha647s-projects.vercel.app/

---

## 📌 Features

- 🔍 Search salons by name or services
- 🗂️ Filter salons based on preferences
- ⭐ View ratings and salon details
- ❤️ Add to Favorites (stored using localStorage)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Clean and attractive UI

---

## 🛠️ Tech Stack

- React.js
- typescript
- HTML5 & CSS3
- React Router
- LocalStorage
